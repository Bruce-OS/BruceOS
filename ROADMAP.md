# BruceOS — Roadmap

---

## v0.1 — Skeleton (Now → 4 weeks)
*Goal: Buildable ISO, even if ugly*

- Fedora 43 base kickstart that actually builds
- CachyOS kernel in COPR
- GNOME + WhiteSur theme applied
- Ghostty + Fish + Starship defaults
- GitHub Actions CI that produces an ISO artifact
- bruceos.com landing page live

## v0.2 — Bruce Walks (4–8 weeks)
*Goal: Usable daily driver for developers*

- `bruce-setup` first-boot TUI working
- Calamares installer with Bruce branding
- Full terminal stack installed (Zellij, Atuin, btop, yazi etc.)
- Ollama running as systemd service
- Basic BruceAI GTK4 app (chat only, no MCP yet)
- Gaming stack installed (Steam, Proton-GE, MangoHud)

## v0.3 — Bruce Thinks (8–12 weeks)
*Goal: AI is actually useful*

- BruceAI with Filesystem + Terminal + Git MCP servers
- GNOME Shell extension overlay (Ctrl+Space)
- First-boot AI wizard (model detection + download)
- Terminal AI functions (`ai`, `explain`, `ask`)
- VFX stack pre-installed (Blender, GIMP, Krita, Kdenlive)

## v1.0 — Bruce Ships (3–6 months)
*Goal: Public release, acquirable*

- Everything in PROGRESS.md v1.0 section ✅
- Offline install works
- Calamares profile selector (Default / Gaming / VFX / Kids)
- `bruce-wine-adobe` packaged and documented
- ISO signed and verifiable
- Download page live on bruceos.com
- Basic docs: README, BUILD, CONTRIBUTING
- GitHub Sponsors / Open Collective for donations

## v2.0 — Bruce Bridges (6–12 months)
*Goal: Adobe CC users can switch*

- Windows Bridge: VFIO + Looking Glass automated setup
- WinApps per-application GNOME integration
- AI-guided hardware checker for Bridge setup
- AT-SPI desktop control for BruceAI
- Voice input (faster-whisper + Piper TTS)
- MCP server marketplace

## v3.0 — Bruce Creates (12–18 months)
*Goal: Creative professionals have no excuses left*

- TRELLIS.2 3D generation service
- Full generative AI creative stack (FLUX, Wan, ACE-Step)
- Education mode (guardrails, curated app set)
- Certified hardware list
- Cloud streaming fallback (Parsec / NICE DCV)
- OEM partnerships (System76?)
